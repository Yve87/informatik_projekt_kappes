display¶

class display.Plotter(isIon=False)¶

Displays specific GA plots with matplotlib.

Imortant note: proper **otherFixedParams must be passed in or the plot will have spooky results

Line styles: ‘’, ‘ ‘, ‘None’, ‘–’, ‘-.’, ‘-‘, ‘:’

Possible Options: xAxis - name of the column to use for the xAxis yAxis - name of the column to use for the xAxis zAxis - name of the column to use for the yAxis

xLabel yLabel zLabel

xLimit yLimit zLimit

Plotter.add_Heatmap(filter, **options)¶

Adds a heatmap to the XY axis plane of the current 3D plot.

Parameters:
  • filter – column object
  • xAxis – the name of the column to use for the x axis
  • yAxis – the name of the column to use for the y axis
  • zAxis – the name of the column to use for the z axis

Note: doesn’t work as expected with realtime update.

Parses the filter for the “result_nevals” column and sums all the values to get the total nevals. Filter must contain “result_nevals” column.
